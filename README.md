# Qatar Export Analysis (2019–2024)

## Project Overview
This project analyzes Qatar’s export performance from 2019 to 2024 using Excel and Power BI.
The objective is to identify export trends, top destination countries, and key product categories.

## Tools Used
- Microsoft Excel
- Power BI
- DAX

## Key Analysis
- Year-wise export value and growth trends
- Top export destination countries
- Product category performance
- Country and product comparison

## Dashboard
Power BI dashboard visuals are included as screenshots and PDF report.

## Outcome
This analysis helps evaluate trade performance and supports data-driven decision-making.
